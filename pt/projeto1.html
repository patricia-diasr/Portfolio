Projeto: Prudente Casa Pet

Sobre o projeto...

Esse projeto foi desenvolvido em grupo como Trabalho de Conclusão de Curso do Técnico de Desenvolvimento de Sistemas. Como cliente, foi escolhido a Prudente Casa Pet, uma empresa de Hospedagem e Day Care de animais de estimação.

O sistema é composto por um site administrativo a ser usado pela dona do estabelecimento, além de um aplicativo com o objetivo de facilitar a comunicação entre a empresa e seus clientes durante a estadia de um pet.

Projeto e Planejamento

Análise de requisitos e planejamento do sistema realizados com diagramas UML, como diagrama de classes, diagrama de sequência e diagrama de atividades.

Desing e Layout

Desenvolvimento de paleta de cores e prototipação de telas com o Figma.

API

API responsável por fazer troca informações entre o front-end e o banco de dados, além da validação de acesso de usuário desenvolvidos com PHP.

Banco de Dados

As informações foram armazenadas em um banco de dados relacional desenvolvido com MySQL, já as imagens e vídeos das postagens foram armazenados no Firebase Storage.

Website

O site foi desenvolido com Angular e Ionic, nele é possível realizar CRUD em informações importantes para o estabelecimento, como pets, tutores, pacotes, procedimentos médicos e pagamentos.

Aplicativo

Também desenvolvido com Angular e Ionic, nele, o administrador consegue fazer postagens referentes aos pets hospedados, que podem ser visualizados por seus tutores.

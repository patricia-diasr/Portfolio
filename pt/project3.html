Projeto: PipocaMania

Sobre o projeto...

O PipocaMania é um aplicativo desenvolvido para aprimorar a experiência de ida ao cinema, tanto para os clientes quanto para os funcionários da rede. A proposta é oferecer uma plataforma completa e moderna, facilitando a administração das sessões e proporcionando ao público uma forma prática e interativa de explorar os filmes em cartaz. O projeto foi criado como parte da disciplina de Projeto de Bloco em Desenvolvimento Front-end com Frameworks.

Para os administradores, o aplicativo permite o cadastro de sessões, visualização do mapa de assentos com possibilidade de bloqueio (para vendas presenciais), e leitura de QR codes de ingressos. Já para os clientes, o PipocaMania oferece recursos como a visualização de filmes em cartaz e próximos lançamentos, detalhes das sessões disponíveis, seleção de assentos, compra de ingressos, avaliação dos filmes assistidos e criação de uma lista de "quero assistir".

Tecnologias utilizadas

  • React Native: Framework utilizado para o desenvolvimento do aplicativo móvel, garantindo uma experiência nativa para usuários Android e iOS.
  • Expo: Plataforma que facilitou o desenvolvimento, testes e publicação do app, além de permitir acesso a recursos como câmera para leitura de QR code.
  • Firebase: Utilizado para armazenar todas as informações da aplicação, como sessões, ingressos, avaliações e listas de interesse.
  • TheMovieDB API: API utilizada para obter informações detalhadas sobre filmes, como sinopse, elenco, trailers e imagens.
  • HTML5 e CSS3: Aplicados na estilização da interface do usuário, assegurando um visual moderno e navegação fluida dentro do aplicativo.

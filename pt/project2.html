Projeto: Littera

Sobre o projeto...

O Littera é um sistema desenvolvido para facilitar a administração de leituras pessoais, oferecendo ao usuário uma plataforma intuitiva e funcional para organizar sua experiência com livros. Criado como parte da disciplina de Projeto de Bloco em Desenvolvimento Back-end, o sistema busca unir praticidade e controle em uma única aplicação.

O sistema permite ao usuário cadastrar livros com capa e informações detalhadas, gerenciar o status da leitura (como "Quero ler", "Lendo" ou "Lido"), registrar datas de início e término, fazer anotações e avaliações com até 5 estrelas, visualizar os livros em formato de vitrine, buscar títulos rapidamente e acompanhar um histórico com estatísticas de leitura, como total de livros lidos e progresso anual.

Tecnologias utilizadas

  • .NET: Framework principal da aplicação, utilizado para estruturar o sistema e fornecer suporte ao desenvolvimento web moderno com alto desempenho.
  • Razor Pages: Utilizado para criar páginas dinâmicas de forma simples e eficiente, unindo a lógica do servidor com o HTML da interface.
  • Entity Framework: ORM (Object-Relational Mapper) que facilitou a comunicação entre o sistema e o banco de dados, permitindo trabalhar com dados por meio de classes C#.
  • SQLite: Banco de dados leve e eficiente, ideal para aplicações locais ou de pequeno porte. Foi utilizado para armazenar todas as informações do usuário e seus livros.
  • HTML5 e CSS3: Utilizados para estruturar e estilizar a interface do usuário, garantindo uma navegação intuitiva e visual agradável.
